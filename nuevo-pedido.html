Peso Volumétrico ℹ El peso volumétrico de un envío es el cálculo que refleja la densidad de un paquete. Un artículo menos denso generalmente ocupa menos volumen de espacio, comparado con el peso real. El peso volumétrico o dimensional se calcula y compara con el peso real de cada pieza del envío para determinar cuál es mayor. El peso mayor se utiliza para calcular el costo total del envío.
